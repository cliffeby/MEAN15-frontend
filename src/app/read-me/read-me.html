Loan & Offers Management System - Project Overview

Project Overview

This system is a full-featured platform for administrators and customers, enabling management of loans, offers, users, and dashboards. It provides secure authentication, dynamic dashboards, and CRUD operations for loans and offers.

Tech Stack & Backend Setup

  • Frontend:Angular 20
  • Backend: Node.js with Express.js
  • Database: MongoDB using Mongoose ODM
  • Authentication: JWT token-based authentication
  • Password Security: bcrypt for password hashing
  • API Endpoints: RESTful endpoints for loans, offers, and authentication
  • Middleware: Helmet for security headers, CORS, rate limiting, compression, and error handling
  • Logging: Morgan for request logging (development and production modes)
  • Deployment Ready: Modular route and controller structure with production-ready error handling

Key Features (End-User Readable)

Dashboard Overview

  • Real-time total loans, total offers, and total customers.
  • Quick overview of loan statuses (Pending, Approved, Declined) via charts.
  • Direct access to loan and offer lists from the dashboard.

Golf Management (CRUD)

  • Create Golf Entry: Admins can add new golf entries with amount, term, interest rate, and status.
  • Read/List Golf Entries: View all golf entries in searchable, sortable, paginated tables.
  • Update Golf Entry: Edit golf entry details or status seamlessly.
  • Delete Golf Entry: Remove golf entries with confirmation prompts.
  • Golf Entry Detail Page: View all golf entry information including customer details.

Offer Management (CRUD with NgRx)

  • Create Offer: Admins can add new offers using a reactive form integrated with NgRx.
  • Read/List Offers: Offers displayed in elegant tables or cards with real-time updates.
  • Update Offer: Edit existing offers; state management handled by NgRx store and effects.
  • Delete Offer: Remove offers with confirmation; NgRx updates state automatically.
  • State Management: Offers CRUD fully integrated with NgRx actions, reducers, selectors, and effects.

Authentication & User Management

  • Secure login/logout for admins and customers.
  • Role-based access: Admins manage all loans and offers; customers view their own data.
  • JWT token-based authentication for API calls.
  • Password hashing with bcrypt for secure storage.

Form Handling & Validation

  • Reactive forms for offers and loans with real-time validation and feedback.
  • Error messages and notifications for invalid input or failed actions.

Interactive Tables & UI

  • Sorting, searching, and pagination for loans and offers.
  • Material tables and cards with responsive layouts.
  • Dynamic filtering by multiple fields (status, amount, customer, offer type).

Notifications & Feedback

  • Success and error notifications on loan and offer actions.
  • Confirmation prompts for critical actions like deletion.

User-Friendly Design

  • Clean, responsive UI using Angular Material components.
  • Toolbar with navigation and action buttons for better workflow.
  • Vertical and card-based forms for better readability.

Additional Notes

  • All components are responsive and enterprise-ready.
  • Supports scaling with additional modules like Reports, Analytics, and Notifications.
  • Seamless integration between CRUD operations and state management via NgRx.
  • Backend is production-ready with secure authentication, proper error handling, and modular design.
